# Capybara Farm Defense

## Описание проекта

**Capybara Farm Defense** — самодостаточная 2D-игра на Python и Pygame. Игрок развивает небольшую ферму капибар, собирает ресурсы, строит защиту, размещает капибар разных классов и отражает волны врагов. Все изображения, персонажи, карта, эффекты и интерфейс рисуются программно средствами Pygame, без внешних ассетов.

## Жанр

Стратегия / защита базы с элементами симулятора фермы.

## Основные механики

- Карта-сетка 16x12.
- Фаза подготовки между волнами: строительство, улучшение и размещение капибар.
- Фаза атаки: враги идут к домику капибар в центре карты.
- Ресурсы: трава, ягоды, дерево и монеты.
- Ресурсы производятся фермой автоматически и появляются на карте как кликабельные бонусы.
- Лучшие прохождения сохраняются в JSON и показываются в таблице рекордов.
- Враги атакуют капибар и постройки, а также пытаются разрушить главный домик.
- Каждая 5-я волна содержит босса.
- После волны игрок получает награду.
- Победа засчитывается после защиты фермы до 15-й волны.

## Типы капибар

- **Защитник** — ближний бой, сбалансированная капибара для ранней обороны.
- **Лучник** — атакует врагов на расстоянии.
- **Строитель** — ускоряет производство ресурсов.
- **Лекарь** — восстанавливает здоровье союзным капибарам.
- **Танк** — много здоровья, блокирует и задерживает врагов.

## Типы врагов

- **Обычный враг** — базовый противник.
- **Быстрый враг** — слабее, но быстрее добирается до базы.
- **Тяжелый враг** — медленный и прочный.
- **Налетчик** — предпочитает атаковать постройки.
- **Босс** — появляется каждую 5-ю волну, имеет много здоровья и высокий урон.

## Постройки

- **Домик капибар** — главная база. Если он разрушен, игра проиграна.
- **Ферма травы** — повышает производство травы.
- **Ягодный куст** — производит ягоды.
- **Деревянная башня** — атакует врагов на расстоянии.
- **Забор** — блокирует и замедляет продвижение врагов.
- **Лечебный пруд** — лечит капибар рядом.

## Управление

- **ЛКМ по карте** — поставить выбранную постройку или капибару.
- **ЛКМ по ресурсу** — собрать ресурс.
- **ЛКМ по панели** — выбрать постройку, капибару или действие.
- **ПКМ по клетке** или **U** — улучшить постройку на клетке.
- **Space** — начать волну досрочно.
- **Esc** — пауза / возврат.
- **F5** — сохранить прогресс.
- **F9** — загрузить прогресс.

## Архитектура проекта

```text
Capybara-Farm-Defense/
├── main.py
├── requirements.txt
├── README.md
└── src/
    ├── __init__.py
    ├── building.py
    ├── capybara.py
    ├── constants.py
    ├── enemy.py
    ├── farm.py
    ├── game.py
    ├── particles.py
    ├── resource_manager.py
    ├── save_manager.py
    ├── ui.py
    ├── utils.py
    └── wave_manager.py
```

Ключевые классы:

- `Game` — главный игровой цикл, состояния меню/игры/паузы/победы/поражения.
- `Farm` — карта, здания, капибары, враги и взаимодействие систем.
- `Capybara` — классы капибар, характеристики и AI.
- `Enemy` — типы врагов, движение, обход препятствий и атака.
- `Building` — постройки, здоровье, улучшения и игровые эффекты.
- `ResourceManager` — ресурсы, производство и собираемые бонусы.
- `WaveManager` — фазы подготовки, спавн волн, награды и боссы.
- `UI` — меню, HUD, панели выбора и подсказки.
- `SaveManager` — сохранение прогресса, рекорда и настроек в JSON.
- `ParticleSystem` — частицы сбора ресурсов и эффекты атак.

## Используемые технологии

- Python 3.10+
- Pygame 2.6+
- JSON для сохранений

## Инструкция запуска

1. Установите зависимости:

```bash
pip install -r requirements.txt
```

2. Запустите игру:

```bash
python main.py
```
